Steam API Setup

To view your Steam games, you'll need to provide your Steam API key and Steam ID64.

Note: This data is processed locally in your browser and not stored on any server.

Troubleshooting: If you get CORS errors, try using a different browser or the manual input option.

How to get your credentials:
• Steam API Key: Visit steamcommunity.com/dev/apikey
• Steam ID64: Visit steamid.io and enter your Steam profile URL
